2Plates

A Wireframe for a CrossFit Companion App

This is a wireframe for a potential iPhone app which I named “2 Plates”. Check it out HERE

The concept for this project was to design a wireframe for a CrossFit Companion which would help users keep track of milestones achieved during their workout sessions. This app is a combination of multiple other apps that are available on the market, as well as containing features that aren’t immediately available on the app store.

For example, it has a tool which allows users to record themselves, and then have the ability to draw on the video. This would be useful for showing weaknesses in a users form during a specific exercise. Another tool allows users to setup competitions called “ThrowDowns”, and also keep track of the progress of the competition.

There are many more features of this app, and I think it does a good job of showing what is possible with a wire framing application.

Some Pics of the Project

Profile Tab

The main tab of the app was the "Profile" tab. This is the homescreen for the user and contains a lot of the relevant information which I thought would be important for users.

The top half of the application has a lot of the information about the user. For example: a profile picture, a custom banner picture, as well as information about which gym the user attends, as well as natural age, and current age.

The Middle of the application has some user data including how many workouts were recorded, how many of those were recent (within a 2 month period), how many Personal Records have been recorded, and how many ThrowDowns the user has participated in. (Note: Throwdowns are custom fitness tournaments that organizations can setup.)


The bottom half of the application has informatino about the latest workout which includes: date of the workout, what the name of the workout was, and what the recorded Time / Score is. It also includes the the navigation bar.

Menu System

I used a classic menu system which should be familiar to most users. The user just taps the 3 lines icon which is located in the upper left, and a menu which was previosly hidden will slide over to take up most of the screen real estate.

History Tab

The history page has a lot of the information you would expect. The main bar up top let's users add a new event, or view media previously recorded (botch pictures & video).

The main part of this window would show a caldendar much like you would see on iOS devices. I envision a type of system that would give the user feedback so he could remember which days he did workouts. Possibly by showing colored dots or some such.

Throwdowns

Throwdowns are a concept that I had been working on with a friend, and I figured that I would throw it in this project just for fun.

The concept of a throwdown is that groups of people can have a system which would allow them to setup competitions. The group would choose a administrator for the throwdown and that person would be responsible for entering scores and other duties as they saw fit (adding and removing individuals etc...)

This part of the app would also have a scoreboard which would show the position of everyone in the competition.

You can even search for other competitions near you!
