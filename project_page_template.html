WWF CO-Lab

Helping to Save Rainforests: One Site at a Time

One of the best experiences I had while in graduate school was the opportunity to pair up with an organization in order to help them accomplish some of their goals. In our case we joined with the World Wildlife Fund, and for a whole semester we would work hand in hand with them in order to create a tool which would be useful to them.

After much dialogue we decided upon a project which would help the WWF help spread the word about FSC products in order to help explain to the public some steps they could employ to help save forests around the world.

If you are interested in seeing the site for yourself and learning about FSC product, feel free to take a look at it HERE

Some Pics of the Project

Intro Slide Mockup

We spent a lot of time going through many different pictures of the WWF database. We ended up choosing this one because we thought the expression of the tiger expressed lots of emotion.

Interactive Floor-Plan Mockup

Our group was dead set on including some interactive elements in the site. The first one we decided on was an interactice floor plan which would allow users to see where FSC related products could be found in their home, office, or school.

Interactive Floor-Plan Mockup

The second interactive element for this project included an interactive map of the US which we hoped would express how much actual area a tiger needs inorder to live, hunt, and mate. For this elements we used a really awesome online tool for visualizations called datawrapper. It was really useful, and added a nice touch to the site.
